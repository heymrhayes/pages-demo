Simple AJAX Call to the NASA APOD API

When this page loads, it's making an AJAX call using the jQuery $.ajax function to query a NASA API. https://api.nasa.gov/api.html 


      var url = "https://api.nasa.gov/planetary/apod?api_key=2sH5ks1PRiOPyzbYwIdJmk0WSqwcSdQ6hNNz2lip";
      $.ajax({
        url: url,
        method: 'GET',
      }).done(function(result) {
        console.log(result);
      }).fail(function(err) {
        throw err;
      });
